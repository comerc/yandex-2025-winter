# 19. Беседа с Големом

| Ограничение времени | 1 с    |
| ------------------- | ------ |
| Ограничение памяти  | 256 МБ |

Преодолев расщелины, Кодерун встречает у ворот замка Снежного Голема, старого прислужника Лорда Нуля. Герой уже готов вступить в схватку, но Голем оказывается безобидным и рассказывает свою историю.

Когда-то Лорд экспериментировал и создавал Големов разных цветов. Всякий раз, когда сталкивались два Голема разных цветов, Лорд приказывал им поменять цвет на наиболее «совершенный». Процесс продолжался, пока все големы не становились одного цвета, наиболее «совершенного» по мнению Лорда.

В лаборатории Лорда находилось $N$ Големов $m$ цветов. Цвета пронумерованы от 1 до $m$, и чем больше номер, тем цвет «совершеннее».

Каждый цикл Лорд выбирал пару разных Големов (равновероятно среди всех неупорядоченных пар). Если цвета совпадали — ничего не происходило. Если цвета различались $i \neq j$, оба Голема превращались в более «совершенный» цвет $\max(i, j)$.

Эксперимент повторялся, пока все Големы не становились одного цвета. Требуется найти ожидаемое количество циклов, необходимых для этого.

## Формат ввода

В первой строке дано целое $m$.

Во второй строке даны $m$ целых неотрицательных чисел $a_1, \ldots, a_m$.

Ограничения на вход: $1 \le m \le 200\,000$, $0 \le a_i$, $\sum_{i=1}^m a_i = N \le 2 \cdot 10^6$, и $N \ge 1$.

## Формат вывода

Требуется вывести такое целое $x$ от 0 до $10^9+7-1$, что $x \equiv P \cdot Q^{-1} \pmod{10^9+7}$, где $T = P/Q$ в несократимой дроби, $Q > 0$, а $Q^{-1}$ — обратный по модулю $10^9+7$.

### Примечание

Когда Кодерун завершает расчёт, Голем благодарит его за беседу и отпускает с миром.

## Пример 1

**Ввод**

```
2
2 0
```

**Вывод**

```
0
```

## Пример 2

**Ввод**

```
2
1 1
```

**Вывод**

```
1
```
