# Ориентирование на местности

Кодерун отрабатывает навык ориентирования в двумерной Долине Синтаксиса. Он начинает движение из точки (0, 0) и выполняет серию команд, проверяя, как меняются расстояния до установленных сигнальных меток.

## Возможные команды

- **N** — движение на север: (x, y) → (x, y + 1)
- **S** — движение на юг: (x, y) → (x, y - 1)
- **W** — движение на запад: (x, y) → (x - 1, y)
- **E** — движение на восток: (x, y) → (x + 1, y)

## Задача

После каждой команды вычисляется сумма манхэттенских расстояний от текущего положения Кодеруна до всех N меток.

Манхэттенское расстояние между двумя точками определяется как сумма модулей разностей их координат. Для точек (x1, y1) и (x2, y2) манхэттенское расстояние равно: |x1 - x2| + |y1 - y2|.

Нужно вывести общую сумму расстояний от Кодеруна до всех меток после каждой команды.

## Формат ввода

В первой строке даны два целых числа N (1 ≤ N ≤ 100 000) и M (1 ≤ M ≤ 300 000) — количество меток и количество команд.

В следующих N строках заданы координаты меток: по два целых числа x и y (|x|, |y| ≤ 1 000 000) в каждой строке.

В последней строке вводится строка из M символов, задающая команды для движения.

## Формат вывода

Выведите суммарное расстояние до всех меток после выполнения каждой инструкции.

## Ограничения

- Ограничение времени: 3 секунды
- Ограничение памяти: 512 МБ

## Примеры

### Пример 1

**Ввод:**

```
1 3
0 -10
ENE
```

**Вывод:**

```
11
12
13
```

### Пример 2

**Ввод:**

```
3 5
0 0
1 1
1 -1
NESSW
```

**Вывод:**

```
5
4
3
4
5
```

### Пример 3

**Ввод:**

```
5 10

-1 0

```

**Вывод:**

```
13
10

14
19
14
11
10

```
