# 15. Метель усиливается

**Решена** | **Сложная**

Кодерун взбирается по крутым склонам хребта в Крае Архитектурных Бурь.

Чем выше он поднимается, тем сильнее искажает цифровое пространство бушующая метель: отдельные текстовые фрагменты алгоритмов разрываются на части, перемешиваются, а затем сходятся обратно в новом порядке.

Чтобы преодолеть очередной участок пути, Кодерун должен разобраться с подобной «перестановкой» над строками.

Кодерун рассматривает строки. Назовём _кодированием_ строки w её разбиение на последовательные подстроки, результатом кодирования — запись этих подстрок в обратном порядке (то есть, сначала идёт последняя, потом предпоследняя и так далее). Два кодирования будем считать различными, если различаются именно разбиения как множества позиций разделения между подстроками (при этом результаты кодирования могут как совпадать, так и отличаться).

Вам дано две строки s и t. Обозначим за f(x, s) для строки x количество кодирований строки s, результат которых лексикографически меньше x.

Ваша задача — найти сумму f(y, s) по всем непустым подстрокам y строки t. Так как ответ может быть большим, найдите его остаток от деления на 998 244 353.

## Формат ввода

Первая строка содержит количество тестовых наборов T (1 ≤ T ≤ 500).

Далее следует описание тестовых наборов:

- Первая строка каждого набора содержит строку s, состоящую из строчных английских букв.
- Вторая строка каждого набора входных данных содержит строку t, также состоящую из строчных английских букв. Длина каждого из слов не превосходит 500.

Гарантируется, что суммарная длина всех строк не превосходит 1000.

## Формат вывода

Выведите одно число — сумму f(y, s) по всем непустым подстрокам y строки t. Так как ответ может быть большим, найдите его остаток от деления на 998 244 353.

## Примечание

Когда вычисление завершено, фрагменты текста вокруг Кодеруна складываются в стабильную конфигурацию и метель чуть отступает, позволяя сделать ещё один шаг вверх, ближе к вершине Пика Кода.
