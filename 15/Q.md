# 15. Снежинки

Метель стихла, можно немного передохнуть. Кодерун смотрит на Пик Кода с уступа, на который он только что вышел из пещеры. С неба, затянутого тёмными облаками, мерно опускаются снежинки. В снегопаде Кодерун видит геометрическую прогрессию.

Эта геометрическая прогрессия `b_i = aq^i` с целыми `a` и `q`. Рассматриваются целые индексы `n, m, k, s ∈ [L, R]`. Нужно найти количество упорядоченных четвёрок `(n, m, k, s)`, для которых

`(aq^n - aq^m) / (aq^k - aq^s)`

то есть частное

`(aq^n - aq^m) / (aq^k - aq^s)`

целое число.

Вместо `aq^0` будем писать просто `a`.

## Формат ввода

Вводится четыре целых числа: `a, q, L, R`.

## Ограничения

- `|a| ≤ 10^9`
- `0 ≤ q ≤ 10^9`
- `-10^18 ≤ L ≤ R ≤ 10^18`
- `N = R - L + 1 ≤ 10^12`

## Формат вывода

Одно целое число — искомое количество четвёрок `(n, m, k, s)` по модулю `10^9 + 7`.

## Примечание

Рассмотрим тест `a = 1, q = 2, L = 0, R = 3`. Тогда всего индексов `N = R - L + 1 = 4`, то есть `n, m, k, s ∈ {0, 1, 2, 3}`, `k ≠ s`.

Если `n = m`, то числитель `2^n - 2^m = 0`, а знаменатель `2^k - 2^s ≠ 0` при любом `k ≠ s`, значит делимость выполняется всегда.

Число таких четвёрок: `4 * 4 * (4 - 1) = 48`.

Теперь случай `n ≠ m`. Всего упорядоченных пар `(n, m)` при `n ≠ m`: `4 * 3 = 12`.

Разобьём их по значению `|n - m|`:

- `|n - m| = 1`: `(0, 1), (1, 0), (1, 2), (2, 1), (2, 3), (3, 2)` — 6 пар
- `|n - m| = 2`: `(0, 2), (2, 0), (1, 3), (3, 1)` — 4 пары
- `|n - m| = 3`: `(0, 3), (3, 0)` — 2 пары

Для каждой группы можно вручную проверить количество допустимых `(k, s)`:

- при `|n - m| = 1` каждая пара даёт 4 подходящих варианта `(k, s)`, всего `6 * 4 = 24`
- при `|n - m| = 2` каждая даёт 6 вариантов `(k, s)`, всего `4 * 6 = 24`
- при `|n - m| = 3` каждая даёт 4 варианта `(k, s)`, всего `2 * 4 = 8`

Итого при `n ≠ m`: `24 + 24 + 8 = 56`.

Общий ответ: `48 + 56 = 104`.

## Пример 1

**Ввод:**

```
1 2 0 3
```

**Вывод:**

```
104
```

- Ограничение времени: `1 c`
