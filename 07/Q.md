# Осколки данных

В глубинах Ледяных пещер Кодеруна обнаружены кристаллы, содержащие замороженные числа. Кодерун находит два параметра: `n`, который определяет длину массива, и `s`, который задает допустимые пределы для сумм подмассивов.

Массив `A = (a_1, a_2, ..., a_n)` из `n` целых чисел называется **замечательным**, если он удовлетворяет двум условиям:

1. **Ограничение на сумму подмассивов:** Для всех подмассивов `sum(l, r)` (где `sum(l, r)` — это сумма элементов от индекса `l` до `r` включительно, для `1 ≤ l ≤ r ≤ n`), сумма должна находиться в диапазоне `[-s, s]`. То есть `-s ≤ sum(l, r) ≤ s`.

2. **Ненулевая сумма подмассивов:** Ни один подмассив массива не может иметь сумму, равную нулю.

## Задача

Подсчитать количество таких "замечательных массивов" для заданных значений `n` и `s`. Результат для каждого теста должен быть выведен по модулю `998244353`.

## Формат ввода

В первой строке вводится одно целое число `T` — количество тестов (1 ≤ T ≤ 5×10^4).

В следующих `T` строках вводятся по два целых числа: `n` и `s` (1 ≤ n, s ≤ 2×10^5).

## Формат вывода

Для каждого теста выведите одно целое число — количество замечательных массивов по модулю `998244353`.

## Ограничения

- Ограничение времени: 2 секунды
- Ограничение памяти: 256 МБ

## Примеры

### Пример 1

**Ввод:**

```
4
2 3
3 2
3 3
1 100
```

**Вывод:**

```
18
0
24
200
```
