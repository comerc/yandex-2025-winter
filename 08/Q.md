# Числа в кристаллах

В глубинах Ледяных пещер Кодеруна обнаружены кристаллы, содержащие замороженные числа. Некоторые из этих чисел реагируют на прикосновение и "разлагаются" на множители, образуя строгие последовательности. Такие числа называются "k-интересными".

Число называется **k-интересным**, если его можно разложить хотя бы одним способом на ровно `k` различных натуральных множителей, все большие 1, таких, что эти множители образуют строго возрастающую последовательность.

Нужно ответить на `q` запросов. Каждый запрос характеризуется тремя целыми числами: `k`, `l` и `r`. Для каждого запроса нужно посчитать количество `k`-интересных чисел в отрезке `[l; r]` (включительно).

## Формат ввода

В первой строке вводится одно целое число `q` — количество запросов (1 ≤ q ≤ 5×10^4).

В следующих `q` строках вводятся по три целых числа: `k`, `l`, `r` (1 ≤ k ≤ 100, 1 ≤ l ≤ r ≤ 700000).

## Формат вывода

Для каждого запроса выведите одно целое число — количество `k`-интересных чисел в отрезке `[l; r]`.

## Примечание

**Первый запрос (1 1 10):** Все числа кроме 1 являются 1-интересными. Они могут быть разложены на 2, 3, 4, 5, 6, 7, 8, 9, 10 соответственно (само число является одним множителем).

**Второй запрос (2 25 25):** Число 25 нельзя разложить на строго возрастающие множители. (25 = 5 × 5, множители не различны).

**Третий запрос (3 30 30):** Число 30 можно разложить на `2 × 3 × 5`. (Это 3 различных, строго возрастающих множителя, больших 1).

## Ограничения

- Ограничение времени: 3 секунды
- Ограничение памяти: 256 МБ

## Примеры

### Пример 1

**Ввод:**

```
4
1 1 10
2 25 25
3 30 30
3 1 100
```

**Вывод:**

```
9
0
1
20
```
