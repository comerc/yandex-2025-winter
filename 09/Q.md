# Испытание Глитчей

Продвигаясь глубже в Ледяные пещеры, Кодерун встречает Снежных Глитчей — слуг Лорда Нуля. Они преграждают путь. Но так как Глитчи — нематериальные сущности, они не атакуют, а требуют пройти испытание. Проход откроется только после успешного решения задачи.

Голос из глубин говорит: "Разложи число на его истинные делители!"

Кодерун видит видение: перед ним целое число `n` (1 ≤ n ≤ 10^9), а также массив `a` из `k` различных целых чисел (1 ≤ k ≤ min(n, 10^5)), где каждое `a_i` удовлетворяет условию 1 ≤ a_i ≤ min(n, 10^6). Пусть `p_i` — i-е простое число в порядке возрастания, при этом `p_1 = 2`. Ваша задача — найти количество натуральных делителей числа `S` и вывести остаток от деления этого количества на `10^9 + 7`.

## Определение числа S

Число `S` определяется по формуле:

```
S = n! / (A * P)
```

где:

- **A** = a_1 × a_2 × a_3 × ... × a_k (произведение всех элементов массива `a`)
- **P** = p_1^α(p_1) × p_2^α(p_2) × p_3^α(p_3) × ...

## Определение α(p)

Показатель степени `α(p)` для простого числа `p` определяется следующим образом:

```
α(p) = { 0,     если p ≤ min(n, 10^6)
       { v(p),  если p > min(n, 10^6)
```

где `v(p)` — показатель степени простого числа `p` в разложении `n!` на простые множители (формула Лежандра).

**Иными словами:** `P` — это произведение всех простых чисел, больших чем `min(n, 10^6)`, возведенных в степень их показателя в разложении `n!`.

## Формат ввода

В первой строке вводятся два целых числа: `n` (1 ≤ n ≤ 10^9) и `k` (1 ≤ k ≤ min(n, 10^5)).

Во второй строке вводятся `k` целых чисел — элементы массива `a` (каждое `a_i` удовлетворяет условию 1 ≤ a_i ≤ min(n, 10^6)).

## Формат вывода

Выведите одно целое число — ответ на задачу (количество натуральных делителей числа `S` по модулю `10^9 + 7`).

## Примеры

### Пример 1

**Ввод:**

```
1 1
1
```

**Вывод:**

```
1
```

**Пояснение:** S = 1! / (1 × 1) = 1. У числа 1 ровно один натуральный делитель: 1.

### Пример 2

**Ввод:**

```
5 3
1 5 1
```

**Вывод:**

```
8
```

**Пояснение:** S = 5! / (1 × 5 × 1) = 120 / 5 = 24. У числа 24 ровно 8 натуральных делителей: 1, 2, 3, 4, 6, 8, 12, 24.

### Пример 3

**Ввод:**

```
3 4
1 2 3 1
```

**Вывод:**

```
1
```

**Пояснение:** S = 3! / (1 × 2 × 3 × 1) = 6 / 6 = 1. У числа 1 ровно один натуральный делитель: 1.
