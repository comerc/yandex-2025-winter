# Задача 03: Распределение экипировки

## Условие

В тренировочном лагере Кодерун распределяет экипировку между командами. Всего есть M единиц снаряжения и N групп альпинистов, каждая из которых требует определённое количество единиц экипировки для своих тренировок.

Однако запас экипировки ограничен — снаряжения не хватает, чтобы полностью укомплектовать все группы. Для каждой группы недостача приводит к проблемам в тренировках. Уровень проблем для группы равен **квадрату недостающего количества единиц экипировки**.

Нужно распределить снаряжение так, чтобы минимизировать суммарное количество проблем.

## Формат ввода

В первой строке вводятся числа M (1 ≤ M < 2 ⋅ 10^9) и N (1 ≤ N ≤ 10^5) — количество единиц снаряжения и групп соответственно.

Во второй строке вводится N чисел W_i (1 ≤ W_i ≤ 2 ⋅ 10^9) — потребность каждой группы в снаряжении. Гарантируется, что суммарная потребность строго больше доступного количества.

## Формат вывода

Выведите минимальное суммарное количество «проблем подготовки» по модулю 10^9 + 7.

## Примеры

### Пример 1

**Ввод:**

```
5 3
1 3 2
```

**Вывод:**

```
1
```

### Пример 2

**Ввод:**

```
10 4
4 5 2 3
```

**Вывод:**

```
4
```

**Примечание:** Разберем второй пример:

- Дано 10 единиц экипировки и 4 группы с потребностями [4, 5, 2, 3].
- Оптимальное распределение будет иметь вид: [3, 4, 1, 2].
- Уровень проблем для такого распределения будет:
  ((4-3)^2 + (5-4)^2 + (2-1)^2 + (3-2)^2) mod (10^9 + 7) = (1 + 1 + 1 + 1) mod (10^9 + 7) = 4

### Пример 3

**Ввод:**

```
1 1
2
```

**Вывод:**

```
1
```

## Ограничения

- Ограничение времени: 2 секунды
- Ограничение памяти: 512 МБ
