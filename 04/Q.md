# Артефакты

Кодерун спускается в Синтаксическую шахту, чтобы изучить секреты старого кода. В шахте находят артефакты.

**Артефакт** — это натуральное число, у которого больше натуральных делителей, чем у любого меньшего натурального числа.

## Задача

Нужно обработать `q` запросов. Каждый запрос характеризуется двумя целыми числами `l` и `r`. Для каждого запроса нужно найти количество артефактов на отрезке `[l; r]`.

## Формат ввода

В первой строке вводится одно целое число `q` (1 ≤ q ≤ 5×10^4) — количество запросов.

В следующих `q` строках вводятся по два целых числа `l` и `r` (1 ≤ l ≤ r ≤ 10^18) — границы отрезка.

## Формат вывода

Для каждого запроса выведите одно целое число — количество артефактов на отрезке `[l; r]`.

## Ограничения

- Ограничение времени: 1 секунда
- Ограничение памяти: 64 МБ

## Примеры

### Пример 1

**Ввод:**

```
3
1 1
5 10
6 8
```

**Вывод:**

```
1
1
1
```

**Пояснение:**

- В первом запросе `1` является артефактом, так как у него один натуральный делитель (само число), а меньших натуральных чисел нет.
- Во втором и третьем запросах единственным артефактом является число `6`, у которого 4 делителя: `1, 2, 3, 6`. Можно показать, что других артефактов в указанном диапазоне нет.
- Число `8` не является артефактом, так как у него тоже только 4 натуральных делителя (`1, 2, 4, 8`), то есть столько же, сколько у `6`.
