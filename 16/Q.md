# 16. Канатная дорога

Нужно восстановить схему канатной дороги с `n` станциями. Схема системы определяется массивом `a` из `n` элементов, где `a_i` означает, что от станции `i` идет направленный кабель к станции `a_i`.

Система старая, и `q` кабелей уже подключены и не могут быть отсоединены. Эти фиксированные соединения задаются двумя массивами `b` и `c` длины `q`, где `a_{b_j} = c_j` для `1 ≤ j ≤ q`.

Нужно определить количество способов завершить схему канатной дороги (т.е. определить оставшиеся `n-q` соединений) так, чтобы выполнялись два требования:

1. Получившийся граф (который не обязательно связный) имеет количество циклов `k` такое, что `l ≤ k ≤ r`.
2. Массив `a` образует перестановку длины `n`. Это означает, что каждая станция должна иметь ровно один исходящий и ровно один входящий кабель, то есть граф будет представлять собой набор непересекающихся циклов.

Итоговый ответ нужно вывести по модулю 998244353.

## Формат ввода

Первая строка содержит целое число `t` (1 ≤ `t` ≤ 5000) — количество наборов входных данных.

Каждый набор входных данных начинается со строки, содержащей четыре целых числа: `n`, `q`, `l`, `r` (1 ≤ `n` ≤ 3000, 0 ≤ `q` ≤ `n`, 0 ≤ `l` ≤ `r` < `n`).

Вторая строка каждого набора содержит `q` целых чисел — массив `b`.

Третья строка каждого набора содержит `q` целых чисел — массив `c`.

Индексы станций 1-индексированные (1 ≤ `b_i`, `c_i` ≤ `n`).

## Формат вывода

Для каждого набора входных данных в отдельной строке выведите одно целое число — ответ на задачу по модулю 998244353.

## Примечание

В первом наборе входных данных есть только 2 возможных интересных массива `a`: `[2, 1, 3]`, `[2, 3, 1]`. В первом варианте два цикла: `1 -> 2 -> 1` и `3 -> 3`. Во втором один: `1 -> 2 -> 3 -> 1`.

Во втором наборе существует единственный интересный массив `a` `[3, 2, 1]`, с двумя циклами.

В третьем наборе можно показать, что интересных массивов `a` нет.

## Пример 1

**Ввод:**

```
4
3 1 0 2
1
2
3 1 2 2
3
1
4 3 0 4
1 2 3
2 1 1
20 5 4 17
4 5 6 12 13
5 6 4 13 14
```

**Вывод:**

```
2
1
0
677226326
```
