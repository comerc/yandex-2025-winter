# Задача 12

Кодеруну поскорее хочется выйти на поверхность в Край Архитектурных Бурь. Но тоннель, ведущий к выходу перекрыт набором дверей с ледяными кодовыми замками. Их, конечно же, установил Лорд Нуль. Каждый замок открывается только тогда, когда на панели вводится число длины `n`, удовлетворяющее строгому системному правилу.

Назовём число длины `n` без лидирующих нулей **чудесным**, если сумма любых трёх последовательных цифр числа является **хорошим** числом.

Вам дано число `n` и набор чисел, которые являются хорошими. Посчитайте количество чудесных чисел длины `n`. Так как ответ может быть большим, выведите его остаток от деления на 998 244 353.

## Формат ввода

Первая строка содержит два целых числа `n` и `m`.

- `n` (длина числа): `3 ≤ n ≤ 10^12`
- `m` (количество хороших чисел): `1 ≤ m ≤ 28`

Следующая строка содержит `m` различных неотрицательных целых чисел `a_i` (`0 ≤ a_i ≤ 27`), которые являются "хорошими числами".

## Формат вывода

Выведите количество чудесных чисел по модулю 998 244 353, чтобы помочь Кодеруну открыть все замки и двинуться дальше.

## Ограничения

- Ограничение времени: 2 секунды
- Ограничение памяти: 256 МБ

## Примеры

### Пример 1

**Ввод:**

```
3 1
15
```

**Вывод:**

```
69
```

### Пример 2

**Ввод:**

```
5 4
27 10 5 7
```

**Вывод:**

```
518
```

## Примечание

Когда расчёт завершён, панели замков вспыхивают. Двери исчезают одна за другой. Кодерун уже чувствует пронизывающий холодный ветер Края Архитектурных Бурь.
