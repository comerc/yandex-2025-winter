# Задача 21. Покрытие точек непересекающимися единичными кругами

## Условие

Дано $N$ точек на плоскости. Необходимо определить, можно ли покрыть все эти точки некоторым количеством (не более $N$) единичных кругов (радиус $R=1$) так, чтобы круги не пересекались (касаться могут).

Если покрытие существует, нужно вывести "YES", количество использованных кругов $K$ и координаты их центров.
Если покрытие невозможно, вывести "NO".

## Формат входных данных

Первая строка содержит целое число $T$ — количество наборов входных данных.
Далее следуют описания $T$ наборов.
Каждый набор начинается со строки, содержащей целое число $N$ ($1 \le N \le 10$) — количество точек.
Следующие $N$ строк содержат координаты точек $x_i, y_i$ (вещественные числа).

## Формат выходных данных

Для каждого набора данных:

- Если решение существует:
  - Вывести строку "YES".
  - Вывести число $K$ — количество кругов.
  - Вывести $K$ строк с координатами центров кругов.
- Если решения не существует, вывести строку "NO".

## Ограничения

- $N \le 10$.
- Круги имеют радиус $R=1$.
- Круги не должны пересекаться (расстояние между центрами $\ge 2$).
- Точка считается покрытой, если расстояние от неё до центра круга $\le 1$.

## Пример

**Входные данные:**

```
2
2
1.1 1.2
-1.0 -0.9
4
1.0 1.1
1.1 1.0
1.5 1.5
0.0 -1.0
```

**Выходные данные:**

```
YES
2
1.1 1.2
-1.0 -0.9
YES
2
1.0 1.0
0.0 -1.0
```

_(Примечание: выходные данные могут отличаться, так как решений может быть несколько)_
