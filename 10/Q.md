# Задача 10: Сеть тоннелей

## Условие

После того как Глитчи рассыпались в пыль, Кодерун выходит в огромный зал. И тут он понимает, что впереди его поджидает сеть тоннелей и других залов, ведущих вглубь горы. Риск заплутать очень высок, поэтому лучше сперва проанализировать доступные маршруты.

Одни тоннели замёрзли полностью, по другим ещё возможно передвигаться. Чтобы продолжить путь, нужно определить, до каких залов можно дойти, если двигаться только по проходимым тоннелям.

Проходимые тоннели представляют из себя неориентированный взвешенный _мультиграф_ из n вершин и m рёбер. Напомним, что _мультиграфом_ называется граф, в котором могут присутствовать кратные рёбра и петли.

Для каждого k (1 ≤ k ≤ n) найдите такое минимальное w, чтобы в графе существовала хотя бы одна вершина u, из которой достижимо k вершин, включая саму вершину u, если перемещаться можно только по рёбрам с весом не более w.

## Формат ввода

Каждый тест состоит из нескольких наборов входных данных. Первая строка содержит количество наборов входных данных t (1 ≤ t ≤ 3 × 10^4). Далее следует описание наборов входных данных.

Первая строка каждого набора входных данных содержит два целых числа — n, m (1 ≤ n, m ≤ 2 × 10^5) соответственно.

Вторая, третья и четвертая строки содержат в себе массивы a, b, c из m целых чисел (1 ≤ a_i, b_i ≤ n, 1 ≤ c_i ≤ 10^9), которые для каждого i (1 ≤ i ≤ m) обозначают ребро между a_i и b_i весом c_i.

Гарантируется, что сумма max(n, m) по всем наборам входных данных не превышает 2 × 10^5.

## Формат вывода

Для каждого набора входных данных в отдельной строке выведите n целых чисел: k_1, k_2, ..., k_n, где k_i равно минимальному w для k = i. Если такого w не существует, k_i = -1.

## Примечание

В первом наборе входных данных можно взять только вершину u = 1, из неё достижима только одна вершина.

Во втором наборе входных данных для k_2, если взять w = 2, граф состоит из единственной петли и трёх вершин, если же взять w = 3, существует ребро между вершинами 1 и 2, а значит за u можно взять 1. Для k_3 и k_4 если взять w = inf, степени вершин 3 и 4 останутся равны 0, а значит нужного w не существует.

В третьем наборе входных данных для k_2 можно взять u = 1, для k_3 и k_4 u может быть равен 3 вершине, из неё достижимо суммарно 4 вершины, для k_5 нужного w не существует.

## Ограничения

- Ограничение времени: 2 секунды
- Ограничение памяти: 128 МБ
