# 17. Защитное поле

Кодерун преодолел часть пути по канатной дороге, но механизмы слишком сильно заледенели. Дальше придется снова идти пешком. Вдалеке возвышается огромный замок из чёрного льда, логово Лорда Нуля. Вдруг герой замечает мерцание в воздухе — это защитное поле. Перед полем расположена консоль: механизм, проверяющий числовые последовательности. Солнечный Процессор уже так близко, нужно взломать защиту!

У Кодеруна есть длина арифметической прогрессии `n`. Нужно выбрать два положительных целых числа `a` и `d`, чтобы обычная прогрессия

`a, a + d, a + 2d, ..., a + (n - 1)d`

имела дополнительное свойство: последовательность сумм десятичных цифр

`S(a), S(a + d), S(a + 2d), ..., S(a + (n - 1)d)`

тоже образует арифметическую прогрессию (разность может быть нулевой или отрицательной).

Каждое из `a` и `d` должно содержать не более `2 * 10^5` цифр. Если подходящих `a` и `d` не существует, выведите `-1`.

## Формат ввода

Одна строка, содержащая целое число `n` (`1 <= n <= 10^4`).

## Формат вывода

Выведите одну строку, содержащую `-1`, либо две строки:

- первая — число `a` (без ведущих нулей),
- вторая — число `d` (без ведущих нулей).

Длины обоих чисел должны не превосходить `2 * 10^5`. Должно выполняться: `a >= 1`, `d >= 1`, а последовательность `S(a + kd)` является арифметической прогрессией для всех `k = 0, ..., n - 1`.

## Примечание

Лорд Нуль наблюдает из замка и зловеще посмеивается...

## Ограничения

- Ограничение времени: `1 c`
- Ограничение памяти: `256 МБ`

## Пример 1

**Ввод:**

```
10
```

**Вывод:**

```
309
9
```
